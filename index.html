Importar backup

Restaura peças a partir de um arquivo JSON. As peças são adicionadas ao catálogo atual — nada é apagado.

Boas práticas

— Exporte o JSON e guarde no Google Drive ou envie para si mesmo por e-mail

— Para trocar de celular, importe o arquivo no novo aparelho

— O CSV abre diretamente no Excel ou Google Sheets

Os dados ficam salvos no Chrome (IndexedDB). Limpar o cache do navegador apaga tudo — faça backups com frequência.
